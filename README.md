 Fraud Detection System:

This is a Fraud Detection System implemented in Python using a Random Forest Classifier.  
The project predicts whether a transaction is fraudulent or not based on historical data.

---

Project Structure

| File / Folder | Description |
|---------------|-------------|
| `fraud_detection_system.ipynb` | Jupyter Notebook with data preprocessing, model training, and evaluation. |
| `fraud_detection_model.pkl` | Trained Random Forest model saved using pickle. |
| `Fraud_Detection_Assignment(Questionories).pdf` | Assignment questions answered. |
| `Fraud_Detection_Questionaries & Jupyter Notebook Structure.pdf` | Project explanation and structure. |
| `Fraud_Detection_System_Shreyash_Patil.pdf` | Detailed report of the system. |
| `Shreyash Patil Resume.pdf` | Resume for reference. |
| `Fraud.csv` | Dataset (not included in repo due to size >100MB). Download separately. |
| `.gitignore` | Ignores large datasets and unnecessary files. |

---

 Dataset:

- The dataset `Fraud.csv` contains historical transaction data.  
- Due to file size (>470 MB), it is not included in this repository. 
- You can download it from: https://drive.google.com/drive/folders/1hrOZKImq-WIuwARu3eCswxyJWKJ9eKBS?usp=drive_link
